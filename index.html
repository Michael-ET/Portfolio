Checkmark

DaRT (Decisions & reporting templates)

TimeSheets

Checkmark Is a document validation system that allows users to peer review, authenticate and process cases by passing cases and attached documents between users. Whilst managing this app I was able to add features, solve bugs and refactor code to create more efficient processes.

TimeSheets is a timekeeping system which allows users to input their hours spent working on particular cases. The system is able to search the company database to pull case data and produce reports for managers. During my time working on this application I was able to solve bugs, assist with new releases and implement new features.

DaRT (decision & reporting templates) is an MS Word addin which allows users to preload document templates by searching for the required case. The user is then able to edit and input sections with ease by selecting options on the sidebar. These options use the SharePoint API to pull prebuilt paragraph data to load into the document. During my time working with this application, I was able to solve bugs, assist with new releases and manage technical debt.
